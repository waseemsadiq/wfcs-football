# WFCS Football Admin Guide

Welcome to the WFCS Football admin area. This guide shows you where everything is and how to manage your football competitions.

## What You Can Do

WFCS Football allows you to:

- Create and manage football seasons
- Organise league competitions with automatic fixture generation
- Run knockout cup tournaments with bracket-style fixtures
- Manage teams and player rosters
- Record match results, scorers, and cards
- Track league standings automatically
- Publish fixtures and results to your website

## Quick Start

1. **Create a Season**: Navigate to Admin > Seasons and create your first season. Set it as active.
2. **Add Teams**: Navigate to Admin > Teams and add at least two teams.
3. **Create Competitions**: Set up leagues or cups in your active season.
4. **Generate Fixtures**: WFCS Football automatically creates fixture schedules.
5. **Record Results**: Enter match scores as games are played.

## Browser Requirements

WFCS Football works best in modern web browsers:

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

JavaScript must be enabled for the dashboard and fixture management features.

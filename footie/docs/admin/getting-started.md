# Getting Started

This guide shows you how to access the admin area and navigate the dashboard.

## Logging In

1. Navigate to your WFCS Football website
2. Add `/login` to the end of the URL (e.g., `https://yoursite.com/login`)
3. Enter the admin password
4. Click **Log in**

The admin password is set during installation. Contact your system administrator if you need the password.

## Understanding the Dashboard

After logging in, you see the Football Fixtures Manager dashboard. This is your control centre for managing all competitions.

### Dashboard Overview

The dashboard displays information about your active season. If no season is set as active, you see a prompt to create one.

**Active Season Summary**

At the top of the dashboard:

- **Season name and status**: Shows which season is currently active
- **Quick action buttons**: Create teams, leagues, cups, or manage seasons
- **Statistics panel**: Displays counts of teams, leagues, and cups in the active season

**Upcoming Fixtures Panel**

The fixtures panel allows you to:

- Select a competition from the dropdown menu
- View upcoming matches grouped by date
- Enter results for matches on or before today's date
- See match times for future fixtures

**Competition Lists**

Below the statistics, you see:

- **League Competitions**: All leagues in the active season with team counts
- **Cup Competitions**: All cups in the active season with team counts

Click any competition name to view its details and standings.

## Navigation

The main navigation menu appears at the top of every admin page:

- **Dashboard**: Return to the main dashboard
- **Seasons**: Manage your football seasons
- **Teams**: Add and edit teams
- **Leagues**: Create and manage league competitions
- **Cups**: Create and manage cup tournaments
- **View Site**: Preview the public-facing website
- **Log out**: Sign out of the admin area

## Your First Season

To get started with WFCS Football:

1. Navigate to **Seasons** in the main menu
2. Click **+ Create Season**
3. Enter a season name (e.g., "2024/25 Season")
4. Set start and end dates
5. Tick **Set as active season**
6. Click **Create Season**

With an active season created, you can now add teams and competitions.

Learn more about [managing seasons](seasons.md).

## Adding Teams

Before creating competitions, add at least two teams:

1. Navigate to **Teams** in the main menu
2. Click **+ Add Team**
3. Enter the team name and optional contact details
4. Select a team colour
5. Add player names (one per line)
6. Click **Create Team**

Repeat this process to add more teams.

Learn more about [managing teams](teams.md).

## Creating Your First Competition

With teams added, you can create a league or cup:

**For a League:**

1. Navigate to **Leagues** > **+ Create League**
2. Enter the league name and select your season
3. Set fixture scheduling (start date, frequency, match time)
4. Select the teams to include (minimum 2)
5. Click **Create League**

**For a Cup:**

1. Navigate to **Cups** > **+ Create Cup**
2. Enter the cup name and select your season
3. Set scheduling defaults
4. Select the teams to include (minimum 2)
5. Click **Create Cup**

WFCS Football automatically generates fixtures when you create a competition.

Learn more about [managing leagues](leagues.md) and [managing cups](cups.md).

## Next Steps

After creating your first competition:

- Navigate to the competition's detail page to view fixtures
- Record match results as games are played
- Check league standings or cup bracket progress
- Add more teams or competitions as needed
